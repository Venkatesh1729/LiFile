LiFile:
This project helps CMAS to maintain Files about Life – namely Birth and Death records.
The screens are developed in PHP and the database is MySQL. We have provided interface in such a way that fields can be entered in English or Tamil; or converted to Tamil after entering in English.
Some fields are additionally provided to capture address data, any diseases at birth; or any specific causes for death.
